# 021: Fracture Theming

> Status: ACTIVE
> Priority: P1 (parallel with 020-code-quality-refactor)
> Type: Content/Data

## Problem Statement

The game currently uses generic fantasy zone names (Beach, Cave, Forest) inherited from BrowserQuest. This doesn't match the "Fracture" narrative - a reality-bending catastrophe that merged dimensions.

## Vision

Transform the world into a post-Fracture landscape where:
- Reality is broken and unstable
- Different dimensions bleed into each other
- The old world exists in corrupted fragments
- Danger increases as you approach the source of the Fracture

## Zone Retheme

| Old Name | New Name | Theme |
|----------|----------|-------|
| Village | The Refuge | Last safe haven, survivors gather here |
| Beach | Shattered Coast | Where reality meets the void, unstable |
| Forest | Glitch Woods | Trees frozen mid-corruption, digital artifacts |
| Cave | The Underdepths | Collapsed infrastructure, echoes of civilization |
| Desert | Null Zone | Where nothing survives, reality erasure |
| Lavaland | The Core Breach | Heart of the Fracture, raw energy |
| Boss | Reality's Edge | Where the Fracture originated |

## Boss Retheme

| Old Name | New Name | Theme |
|----------|----------|-------|
| Giant Crab | The Harvester | Mutated scavenger from merged oceans |
| Goblin Warlord | Glitch Stalker | Corrupted hunter, phases in/out |
| Bone Dragon | The Forgotten | Echo of creatures that no longer exist |
| Sand Wurm | Null Devourer | Entity from the void between realities |
| Demon Lord | Core Guardian | Elemental born from the breach |
| Skeleton King | The Architect | Entity that caused/emerged from the Fracture |
